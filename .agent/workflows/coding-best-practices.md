# Apply Agent Skills

1. **Context Loading**:
   - Before writing or modifying code, ALWAYS read `client/agent-skills/skills/react-native-best-practices/SKILL.md`.
   - Identify the relevant category (e.g., Performance, Bundle Size, UI).

2. **Specific Skill Retrieval**:
   - If the task involves Lists, search/read `js-lists-flatlist-flashlist.md`.
   - If the task involves Animations, search/read `js-animations-reanimated.md`.
   - If the task involves Native Modules, search/read `native-turbo-modules.md`.

3. **Implementation**:
   - Apply the specific patterns defined in the skill files.
   - Avoid "Common Pitfalls" listed in the skills.

4. **Verification**:
   - Ensure the new code complies with the project's standard structure.
